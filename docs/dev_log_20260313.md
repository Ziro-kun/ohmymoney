## 2. 핵심 기능 개발 (Core Features)

- **Fixed Expense(고정 지출) 엔진**:
  - **가계부 연동**: `flow.tsx` 내 거래 항목에 `isFixed` 토글 필드 추가
  - **계산 로직 통합**: `useFinanceStore.ts`의 `calculateDailyBurn` 함수를 전면 개편하여 수동 고정 지출 항목뿐만 아니라 거래 내역(`transactions`) 중 고정 지출로 마크된 항목을 월 단위(30.44일) 가중치로 합산하도록 구현
- **자산 관리(Stock) 모듈 안정화**:
  - 자산/부채 CRUD 기능 완성 (`addAsset`, `updateAsset`, `deleteAsset` DB 연동)
  - 현재 순자산을 자산 합계에서 부채 합계를 빼는 방식으로 정확하게 산출
- **AI 맞춤 전략 시스템**:
  - 사용자 번 레이트와 순자산 비율(Severity Ratio)을 분석하여 3단계(보통, 주의, 심각) 전략 메시지 출력
  - 리브랜딩된 톤앤매너에 맞춰 AI 제안 텍스트 개인화
