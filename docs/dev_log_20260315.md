# 상세 개발 로그 (Expanded Development Log) - 20260315

## 1. 정교한 번레이트 엔진 (Precision Burn Rate Engine)

- **정밀도 향상 (Milli-won Scaling)**:
  - 부동 소수점 오차 누적을 방지하기 위해 모든 번레이트 계산에 `SCALE = 1000` (milli-won) 단위를 도입.
  - 초당 잔액 변화(Ticker) 시뮬레이션의 정확도를 비약적으로 높여 장시간 앱 실행 시에도 데이터 정합성을 유지.
- **가중치 기반 변동 지출 반영**:
  - 고정 지출뿐만 아니라 최근 7일간의 **비정기적 변동 지출(Variable Expenses)**의 평균값을 일일 번레이트에 합산하는 로직 구현.
  - 사용자의 실제 소비 패턴이 반영된 더 현실적인 '미래 프로젝트' 제공.
- **동적 자산 상태 연동**:
  - 대출 상환금이 납부될 때마다 실시간으로 줄어든 원금을 기준으로 이자(번레이트)가 즉시 재계산되도록 `depreciatedAssets` 연동 구조 개선.

## 2. 자동화 및 가상 거래 (Automation & Virtual Transactions)

- **정기 거래 자동 확장 (Virtual Transaction Generation)**:
  - `Transaction` 모델에 `recurringDay` 필드 추가.
  - 사용자가 매월 특정 날짜를 지정하면, 해당 날짜에 맞춰 과거부터 현재까지의 가상 거래(Virtual Transactions)를 실시간으로 생성하여 잔액에 반영.
  - 실제 DB를 복잡하게 만들지 않으면서도 고정 지출의 흐름을 정확히 시뮬레이션.

## 3. 대시보드 UI/UX 고도화 (Dashboard UX)

- **실시간 강조 모드 (Emphasis Mode)**:
  - `react-native-reanimated`를 활용하여 '순자산(Net Worth)'과 '고정 지출(Burn Rate)' 중 원하는 지표를 스와이프 한 번으로 강조할 수 있는 인터랙티브 UI 구현.
  - 강조된 지표에 따라 크기, 불투명도, 위치가 자연스럽게 전환되어 몰입감 있는 경험 제공.
- **데이터 시각화 폴리싱**:
  - 번레이트가 발생 중일 때 빨간색 점이 깜빡이는 `pulseDot` 애니메이션 추가로 시각적 경각심 부여.
  - AI 제안 카드(AICard)의 로직을 고도화하여 30일간의 기회비용 손실액을 복리 기반으로 계산하여 출력.

## 4. 플랫폼 최적화 및 안정성 (Platform & Stability)

- **Android 렌더링 아티팩트 해결**:
  - 일부 안드로이드 기기에서 카드 테두리에 잔상이 남는 문제를 해결하기 위해 `borderWidth`를 전면 제거하고 배경색 면 분할(Solid Filling) 방식으로 디자인 시스템 전환.
- **테마 동기화 및 성능**:
  - 다크/라이트 모드 전환 시 스타일 누락을 방지하기 위해 `makeStyles` 패턴을 모든 주요 컴포넌트에 적용.
  - `app/(tabs)` 내 모든 주요 화면(`flow`, `stock`, `settings`)의 레이아웃 안정화.

## 5. 금융 관리 마법사 고도화 (Wizard Enhancements)
