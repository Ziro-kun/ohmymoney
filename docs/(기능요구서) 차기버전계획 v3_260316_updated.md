## 세부 내용 및 계획 구체화

### 5. 자산 마법사 및 대출 로직 고도화

#### 5.1 대출 등록 프로세스 개선 [✅]
- 대출 항목 생성 후 상환금(원리금) 등록 여부 옵션화 완료.
- "나중에 등록하기" 기능을 통한 허들 감소.

#### 5.2 대출 상환액 자동 계산 (Cascade 로직) [✅]
- `대출원금`, `대출이율`, `대출기간` 기반 자동 상환액 산출 엔진 구현 완료.
- 수동 입력 모드 전환 및 데이터 구조 설계 완료.

### 6. 지출 관리 및 UI 편의성 개선

#### 6.1 정기 지출 발생 시점 정교화 [✅]
- **가상 거래 엔진**: 매월 지정된 날짜(`recurringDay`)에 맞춰 과거/현재의 지출 흐름을 시뮬레이션하여 잔액에 실시간 반영.
- **로직**: 실제 DB를 오염시키지 않으면서도 번레이트에 즉각적인 영향을 주도록 설계.

#### 6.2 날짜 선택 UI (DatePicker) 도입 [✅]
- Native Calendar Picker 연동을 통해 입력 오류 방지 및 직관성 확보.

### 7. 서비스 수익 모델 및 데이터 정책 (Monetization)

#### 7.1 계층별 기능 및 운영 정책
- **무료 플랜 (Free Tier)**: 로컬 DB 기반, 수동 입력, 개인 클라우드 백업, 광고 노출.
- **유료 플랜 (Pro Tier)**: 전용 클라우드 동기화, 자동 인식(SMS/카톡), 마이데이터 연동(검토), 광고 제거, 프리미엄 테마.

---

### 8. [신규] 설정화면 및 데이터 관리 고도화 (🔴)

#### 8.1 시스템 및 테마 설정
- **기능**: 다크/라이트 모드 수동 전환, 금액 단위(원/천원) 설정, 라이브 틱 주기(0.1s~1s) 조절.
- **UI**: 슬라이더 및 세그먼트 컨트롤을 통한 직관적인 세부 조정.

#### 8.2 생체 보안 및 개인정보 보호 (🟢)
- **기능**: 앱 진입 시 FaceID/TouchID 또는 6자리 핀번호 인증. 
- **프라이버시**: '금액 숨기기' 모드 전환 시 메인 화면의 숫자를 마스킹 처리.

#### 8.3 클라우드 동기화 엔진 (🔴)
- **구조**: SQLite DB 암호화 후 전용 Cloud Bucket에 업로드. 
- **동기화 시점**: 앱 종료 시 또는 수동 버튼 클릭 시 백그라운드 동기화 수행.

---

### 9. [신규] 통계 리포트 및 시각화 (🔴) [✅]

#### 9.1 카테고리별 소비 분석
- **시각화**: 파이 차트 및 가로 막대 그래프로 이번 달 지출 상위 5개 항목 표시.
- **필터링**: 주간/월간/연간 단위 전환 및 특정 카테고리만 보기 기능.

#### 9.2 순자산 및 번레이트 추이
- **Trend Line**: 자산 가치 변화와 지출액 변화 곡선을 겹쳐서 표시하여 '적자/흑자 전환점' 가시화.

---

### 10. [신규] 고급 번레이트 및 자동화 (🟡/⚪)

#### 10.1 지능형 번레이트 모니터 (Burn Monitoring) (🟡)
- **Safe Spending**: 현재 순자산이 소진될 때까지 걸리는 시간(Days to Bankruptcy)을 실시간으로 계산하여 경고 모드 제공.
- **목표 연동**: "오늘 치킨을 참으면 자산 목표 달성이 2시간 빨라집니다"와 같은 동기부여 알림.

#### 10.2 자동 입력 (AI 파싱) (⚪)
- **안드로이드**: SMS 권한 획득 후 문지 메시지 내 금액, 업체명, 날짜 자동 추출.
- **iOS/Common**: 클립보드 자동 감지(금액 복사 시 앱 진입 시 팝업 제안) 및 푸시 알림 인터셉트 연구.

---

### 11. [신규] AI 모델 및 감가상각 가이드 (🔴/⚪)

#### 11.1 자산 가치 세이브 예측 모델 (🔴)
- **목표**: 사용자 평균 번레이트와 자산 수익률을 결합하여 1년 후 예상 자산 시뮬레이션.
- **엔진**: TensorFlow Lite 또는 ONNX Runtime 기반 온디바이스 학습 엔진 탑재 시도.

#### 11.2 감가상각 UI 및 로직 정교화 (⚪)
- **차량/부동산**: 제조사, 연식 입력 시 중고차 시세 API 또는 표준 감가율을 연동하여 매월 자산 가액을 자동 하향 조정.
- **반영**: 번레이트와 별개로 '자산 가치 소멸율'을 대시보드에 표시.
